# 🤖 AI Stack Guide

Интерактивный каталог ИИ-инструментов с фильтрами, навигацией и встроенным помощником.

🌐 **Опубликованная версия:** [https://твой-ник.github.io/ai-tools-site/](https://твой-ник.github.io/ai-tools-site/)

## 🚀 Запуск
Сайт работает без сервера. Достаточно открыть `index.html` в браузере или перейти по ссылке выше.

## 🛠 Технологии
- HTML5 / CSS3 / Vanilla JS (ES6+)
- Относительные пути (полная совместимость с GitHub Pages)
- Правило-база чат-бота
- Адаптивная вёрстка
